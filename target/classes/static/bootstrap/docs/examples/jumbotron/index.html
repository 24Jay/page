观于海者难为水

孟子曰：孔子登东山而小鲁，登泰山而小天下。
故观于海者难为水，游于圣人之门者难为言。
观水有术，必观其澜。日月有明，容光必照焉。
流水之为物也，不盈科不行；君子之志于道也，不成章不达。
